#### 如何避免死锁

1. 尽量避免使用多个锁，并且只有需要时才持有锁。
2. 尽量设计好锁的获取顺序
3. 使用带超时的方法，为程序带来更多可控性。
   类似 Object.wait(…) 或者 CountDownLatch.await(…)。
4. 通过静态代码分析（如 FindBugs）
