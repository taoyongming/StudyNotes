#### 第一章 入门

| 项 目      | UNIX    | Windows     |
| ---------- | ------- | ----------- |
| 目标模块   | func.o  | FUNC.OBJ    |
| 静态函数库 | lib.a   | LIB.LIB     |
| 程序       | program | PROGRAM.EXE |

共享库在许多方面类似于Windows中使用的动态链接库。.so库对应于.DLL文件，它们都是在程序运行时加载，而.a库类似于.LIB文件，它们都包含在可执行程序中。
